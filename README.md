# My Skills

A collection of Claude Cowork skills.

## Skills

### executive-storytelling
Craft executive speeches, presentations, all-hands talks, board updates, investor pitches, and keynotes using proven storytelling frameworks (ABT, 3-Act Structure, Golden Circle, STARR, Monroe's Motivated Sequence, Hero's Journey).

**Triggers on:** speech, talk, presentation, keynote, all-hands, board update, town hall, investor pitch, strategy rollout, team address.

**Output:** Slide deck outline + full speech draft with speaker notes, labeled by framework.

## Installing a skill

1. Download the `.skill` file from the [Releases](../../releases) page
2. In Claude Cowork, click "Save skill" to install
